# 🚀 Quick Start Guide

## Get Running in 3 Steps!

### Step 1: Install Dependencies
```bash
npm install
```
⏱️ Takes 2-5 minutes

### Step 2: Start the App
```bash
npm start
```
🌐 Opens automatically at http://localhost:3000

### Step 3: Enable Notifications
Click "Allow" when browser asks for notification permission

---

## ✨ Features to Try

### 1. Download Course for Offline Access
- Click any course card
- Press "📥 Download for Offline"
- Badge appears: "Available Offline"

### 2. Set Course Reminder
- Open a course
- Choose reminder time
- Get notified later!

### 3. Build Your Streak
- Come back daily
- Watch your streak counter grow
- Earn badges at 7, 30, and 100 days

### 4. Complete a Course
- Click "Mark as Complete"
- Enjoy the confetti celebration! 🎉

### 5. Search & Filter
- Type in search bar
- Click tag buttons
- Find courses instantly

### 6. Resume Video Watching
- Start watching a video
- Leave and come back
- Click "Resume Watching"

## 🆘 Need Help?

**Issue:** Port 3000 in use  
**Fix:** `PORT=3001 npm start`

**Issue:** Dependencies error  
**Fix:** `rm -rf node_modules package-lock.json && npm install`

**Issue:** White screen  
**Fix:** Check browser console (F12) for errors
